What is UAT Testing?

User Acceptance Testing (UAT) is when actual users test the application.
The testers write real-life scenarios according to the business and functional requirements.

I use endtest to write each test case. Endtest is an automated cloud testing platform. Endtest includes assertions, so we will be able to apply Pass/Fail criteria on functionality.
